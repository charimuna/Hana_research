# PDFinstall フォルダ - コード説明書

## 概要
このフォルダには、Mobacal（医療情報システム）から患者のPDF文書をダウンロード・抽出するための3つのPythonスクリプトが格納されています。これらのスクリプトは一連の流れで連携して、医療データの自動処理を実現します。

---

## スクリプト構成と処理フロー

```
[1] generate_mobacal_list.py
        ↓
   mobacal_pdf_list.tsv（文書リスト）
        ↓
[2] download_mobacal_pdfs.py
        ↓
   out_pdfs/（ダウンロード済みPDF）
        ↓
[3] extract_pdf_to_csv.py
        ↓
   pdf_summary.csv（抽出データ）
```

---

## 1. generate_mobacal_list.py

### 目的
Mobacal（医療情報サーバー）にログインしてPDF文書の一覧を取得し、ファイル名とURLをTSV形式で保存する。

### 主な機能
- **Webブラウザの自動操作**: Playwriteを使用してChrome自動制御
- **ユーザーのChromeプロファイル利用**: 既存のクライアント証明書設定を活用可能
- **文書一覧の抽出**: Mobacal内の指定期間のPDF文書一覧を自動スクレイピング
- **日付変換**: 様々な日付形式（「12/25(金)」など）をYYYYMMDD形式に統一

### 出力ファイル
- **mobacal_pdf_list.tsv**: 以下の形式で保存
  ```
  YYYYMMDD_患者ID<TAB>文書URL
  例: 20250115_150042	https://s2.movacal.net/24.5/?patient_id=150042&docs_id=12345
  ```

### 使用方法
```bash
python generate_mobacal_list.py
```
- 実行時にブラウザが起動し、手動ログインが必要な場合があります
- ログイン完了後、スクレイピングが自動で進行します

### 処理対象期間
- 開始日: 2015年4月1日（固定）
- 終了日: 実行当日（自動更新）

## 2. download_mobacal_pdfs.py

### 目的
TSVファイルに記載されたURLからPDFをダウンロードし、整理されたファイル名で `out_pdfs/` フォルダに保存する。

### 主な機能
- **複数の入力形式に対応**: 以下のTSV形式を自動判別
  - `filename.pdf<TAB>url`
  - `YYYYMMDD_ID<TAB>url`
  - `YYYYMMDD<TAB>ID<TAB>url`
  - `ID<TAB>url`
  - 自動検出（データから日付とIDを抽出）

- **ロバストな処理**
  - 複数のエンコード形式に対応（UTF-8, Shift_JIS, UTF-16など）
  - タブ区切りとスペース区切り混在への対応
  - 不完全なPDFの検出と無視

- **PDFビューア対応**: インラインビューア、iframe、オブジェクト埋め込みからのダウンロード対応

- **ファイル名の正規化**: 禁則文字をアンダースコアに置換、自動的に`.pdf`拡張子を追加

### 出力ファイル
- **out_pdfs/**: ダウンロードされたPDFファイル
  - ファイル名形式: `YYYYMMDD_患者ID.pdf`
  - 例: `20250115_150042.pdf`

### 使用方法
```bash
python download_mobacal_pdfs.py
```
- 処理開始時にブラウザが起動
- ユーザーのChromeプロファイルに保存されたクライアント証明書を利用
- 手動ログインが必要な場合はプロンプトに従う

### 設定項目（スクリプト冒頭を編集）
- `USER_DATA_DIR`: Chromeプロファイルの場所
- `BASE_ORIGIN`: MobacalサーバーのURL（例: https://s2.movacal.net/24.5/）

## 3. extract_pdf_to_csv.py

### 目的
ダウンロード済みのPDFファイルからテキストを抽出し、患者情報と共にCSV形式で整理する。

### 主な機能
- **PDF解析**: pdfplumberライブラリを使用してPDFからテキスト抽出
- **ファイル名のパース**: `YYYYMMDD_患者ID.pdf` 形式から情報を自動抽出
- **マルチページ対応**: PDF内全ページのテキストを結合して処理
- **日付フォーマット**: YYYYMMDD形式をYYYY-MM-DD形式に変換
- **エラーハンドリング**: 処理失敗時もログを出力して続行

### 出力ファイル
- **pdf_summary.csv**: UTF-8-SIGエンコーディング（Excelで正しく開ける）
  - 列: `患者ID`, `作成日`, `書類名`, `内容`
  
  例:
  ```
  患者ID,作成日,書類名,内容
  150042,2025-01-15,診療記録,"体温：36.5℃、血圧：130/80..."
  150043,2025-01-14,処置報告書,"処置内容：..."
  ```

### 使用方法
```bash
python extract_pdf_to_csv.py
```
- `out_pdfs/` フォルダ内の全PDFを処理します
- 出力CSVのアンコード: `utf-8-sig`（BOM付き、Windows/Excelで安全に開ける）

## 実行シナリオ

### シナリオ1: 新規にデータを取得する場合
```bash
# ステップ1: Mobacalから文書リストを生成
python generate_mobacal_list.py
# → mobacal_pdf_list.tsv が作成される

# ステップ2: リストのURLからPDFをダウンロード
python download_mobacal_pdfs.py
# → out_pdfs/ にPDFが保存される

# ステップ3: PDFからデータを抽出
python extract_pdf_to_csv.py
# → pdf_summary.csv が作成される
```

### シナリオ2: 既存の文書リストからダウンロードのみ
```bash
# ステップ2と3のみを実行
python download_mobacal_pdfs.py
python extract_pdf_to_csv.py
```

### シナリオ3: ダウンロード済みのPDFのみ再抽出
```bash
# ステップ3のみを実行
python extract_pdf_to_csv.py
```

## トラブルシューティング

| 問題 | 原因 | 対処方法 |
|------|------|--------|
| ログインできない | クライアント証明書が設定されていない | Chromeでログイン後、認証情報を保存 |
| PDFがダウンロードできない | URLが無効または期限切れ | リスト生成から再実行 |
| テキストが抽出されない | PDFがスキャン画像形式 | OCR処理が必要（別途対応） |
| CSVに日本語が文字化けする | エンコーディング問題 | ファイルを「UTF-8」で開く |
| ファイル名に特殊文字が含まれる | 禁則文字が元のデータにある | 自動的にアンダースコアに置換される |

## 注意事項

1. **認証情報の安全性**: Chromeプロファイルに保存されたクライアント証明書を使用するため、セキュアな環境で実行してください
2. **実行時間**: 大量のPDFダウンロードの場合、時間がかかります
3. **ネットワーク**: MobacalサーバーへのアクセスにはVPNが必要な場合があります
4. **バージョン管理**: Mobacalのバージョンが変更された場合、URLやレイアウト情報の更新が必要です
